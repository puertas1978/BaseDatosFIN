Ahora que ya sabes lo que es una base de datos, conoce lo que es un Sistema Gestor de Base de Datos.  ¿Podrías definirlo antes de seguir leyendo? ¡Venga, inténtalo!

Sistema Gestor de Base de Datos

Un Sistema gestor de Base de Datos es el programa que administra y gestiona la información que contiene una base de datos, es decir, el Software que utilizamos para poder gestionar y administrar la información que contiene una base de datos. 

¿Qué son los Sistemas Gestores de Bases de Datos?

Comprueba lo que has aprendido

Indica si las siguientes afirmaciones son verdaderas o falsas. 

Question 1

1. Oracle, MySQL, MS Access, son bases de datos

Feedback

False

Todos estos son Sistemas Gestores de Base de Datos, que nos ayudan gestionar y tratar la información existente en una base de datos. 

Question 2

2. Los sistemas gestores de base de datos son software

Feedback

True

Son programas (software) que instalamos para poder gestionar nuestras bases de datos. Normalmente se instalan en servidores, pero nosotros vamos a instalar la versión escritorio en nuestro ordenador. 

Question 3

3. Access es un Sistema de Gestión de Base de Datos de escritorio

Feedback

True

Si, es una herramienta de escritorio que ofrece Microsoft. 

Question 4

4. El lenguaje con el que se opera en bases de datos es SQL

Feedback

True

Es correcto, SQL es como se le denomina al lenguaje que utilizamos para operar en una base de datos.

Question 5

5. Un Sistema Gestor de Base de Datos, solo puede gestionar una base de datos

Feedback

False

Puede gestionar más de una base de datos con reglas y datos diferentes.
