2.5. Challenge 3. Clients Database

Cients Database

Duración:
Dos semanas
Participantes:
Por parejas

Previous tasks 

Task 1. Reading: https://cincodias.elpais.com/cincodias/2020/08/21/companias/1598015157_845688.html   Make a list of companies that appear in it and provide access to their website in the international projects section. 

Phases

Phase 1: We create a relational database

Following the example shown in the notes of this material, you should create a database with 2 potential clients of the companies identified in the previous task. The attributes and entities of the database will be as follows:

  • Client Name
  • Address
  • Country
  • Phone
  • Email
  • Website
  • Contact Person (LinkedIn) and Position
  • Payment Conditions

Phase 2. Update database

Update the position of 3 clients included in the previous database.
