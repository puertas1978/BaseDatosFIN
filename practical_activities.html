30:00

Siguiendo las indicaciones dadas, dibuja usando esta herramienta digital (https://jorgesanchez.net/ermaker/) el modelo Entidad/ Relación para el diseño de una base de datos.  

“Una empresa vende productos a varios clientes. Se necesita conocer los datos personales de los clientes (nombre, apellidos, dni, dirección y fecha de nacimiento).Cada producto tiene un nombre y un código, así como un precio unitario. Un cliente puede comprar varios productos a la empresa, y un mismo producto puede ser comprado por varios clientes. Los productos son suministrados por diferentes proveedores. Se debe tener en cuenta que un producto sólo puede ser suministrado por un proveedor, y que un proveedor puede suministrar diferentes productos. De cada proveedor se desea conocer el NIF, nombre y dirección”.

Tarea 2. Mi primer encargo como gestor/a de Base de Datos

Duración:
30:00
Agrupamiento:
1

Imagina que estás trabajando en el diseño de una base de datos para una empresa de automoción que busca expandirse internacionalmente. La empresa necesita gestionar información sobre las ayudas disponibles para la internacionalización y las entidades que las otorgan. Tu tarea es crear un modelo entidad-relación (E/R) para esta base de datos.

Identifica las Entidades:

Ayudas a la Internacionalización
Entidades Otorgantes
Identifica los Atributos de cada Entidad:

Ayudas a la Internacionalización:

ID de Ayuda (identificador único)
Nombre de la Ayuda
Descripción
Monto de la Ayuda
Fecha de Vigencia
Entidades Otorgantes:

ID de Entidad (identificador único)
Nombre de la Entidad
País de Origen
Tipo de Entidad
Contacto (nombre, correo electrónico, teléfono)
Define las Relaciones entre las Entidades:

Cada ayuda a la internacionalización es otorgada por una entidad otorgante.
Una entidad otorgante puede otorgar múltiples ayudas a la internacionalización.
Representa el Modelo Entidad-Relación en un diagrama:

Utiliza símbolos adecuados para representar las entidades, atributos y relaciones.
Indica la cardinalidad de las relaciones.
Finaliza el Modelo E/R:

Agrega cualquier restricción adicional necesaria, como claves primarias y foráneas.
¡Resuelve este ejercicio diseñando un modelo entidad-relación claro y preciso para la base de datos de ayudas a la internacionalización de la empresa de automoción!
